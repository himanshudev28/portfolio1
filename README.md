# Himanshu Singh — Portfolio

Award-winning personal portfolio website built with React, Three.js, and Framer Motion.

**Live site:** -https://himanshudev1.vercel.app/

---

## Tech Stack

| Layer | Tools |
|-------|-------|
| Framework | React 18 + TypeScript + Vite 5 |
| 3D | React Three Fiber · @react-three/drei · Three.js |
| Animation | Framer Motion |
| Styling | Tailwind CSS v3.4 |
| Fonts | Syne · Inter · JetBrains Mono |

## Features

- Procedural 3D robot built entirely from Three.js primitives — no external model files
- Mouse-tracked head and body rotation with spring lerp
- Coral-rose design system (`#F43F5E`) on pure charcoal — deliberately non-AI palette
- Scroll-triggered section reveals via Framer Motion `whileInView`
- Seamless tech marquee strip with CSS ticker animation
- Fully responsive — mobile-first, tested down to 375 px
- Code-split 3D canvas via `React.lazy` + `Suspense`

## Sections

Hero → Tech Strip → Projects → Experience → About & Skills → Achievements → Contact

## Getting Started

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # production build → dist/
```
